# ADR-005: 数据格式 — Q8.8 定点

## 状态

已接受

## 背景

竞赛要求使用 Q8.8 定点格式 (16-bit 有符号)。需要确定内部计算精度。

## 决策

- I/O: Q8.8 (16-bit 有符号)
- 累加: 40-bit 定点
- Softmax: Q8.8 查表

## 理由

| 格式 | 精度 | 面积 | 选择 |
|------|------|------|------|
| FP16 | 高 | 大 | |
| INT8 | 低 | 小 | |
| Q8.8 | 中 | 中 | **选择 (竞赛要求)** |

- Q8.8 满足竞赛要求
- 40-bit 累加防止溢出
- 查表使用 Q8.8 格式，简化设计

## Q8.8 格式说明

```
[sign(1)] [integer(7)] [fraction(8)]
范围: [-128, +127.99609375]
精度: 1/256 ≈ 0.00390625
```

## 后果

- 输入/输出: 16-bit
- 内部累加: 40-bit
- 面积适中
- 精度满足 mean_abs_error <= 0.03 要求
