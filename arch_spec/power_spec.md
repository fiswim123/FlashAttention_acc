# FlashAttention 加速器 IP — 电源架构

## 1. 电源域划分

### 1.1 电源域定义

| 域名 | 电压 | 覆盖模块 | 说明 |
|------|------|----------|------|
| VDD_CORE | 0.70V | 全部逻辑 | 核心逻辑电源 |
| VDD_MEM | 0.70V | SRAM | 存储器电源 |
| VDD_IO | 0.70V | IO Ring | IO 电源 (AXI 接口) |

本设计简化为单一电源域, 所有模块共享 VDD_CORE。

### 1.2 电源域边界

| 边界 | Isolation | 说明 |
|------|-----------|------|
| CORE ↔ IO | 无 (同域) | 同一电压域 |
| CORE ↔ MEM | 无 (同域) | 同一电压域 |

## 2. 功耗估算

### 2.1 动态功耗

**公式**: P_dynamic = α * C * V^2 * f

| 参数 | 值 | 说明 |
|------|-----|------|
| α (活动因子) | 0.15 | 典型值 |
| C (等效电容) | ~50pF | 估算值 |
| V (电压) | 0.70V | ASAP7 典型 |
| f (频率) | 50MHz | 目标频率 |

**计算**: P_dynamic = 0.15 * 50e-12 * 0.70^2 * 50e6 ≈ 0.184 mW

考虑实际活动因子和毛刺, 估算实际功耗:

| 模块 | 活动因子 | 功耗估算 |
|------|----------|----------|
| fa_ctrl | 0.10 | 0.5 mW |
| fa_dma | 0.20 | 2.0 mW |
| fa_systolic | 0.30 | 8.0 mW |
| fa_softmax | 0.25 | 3.0 mW |
| fa_divider | 0.15 | 1.0 mW |
| fa_buffer_mgr | 0.20 | 2.0 mW |
| fa_regfile | 0.05 | 0.5 mW |
| SRAM | 0.15 | 5.0 mW |
| **总计** | | **~22 mW** |

### 2.2 静态功耗 (Leakage)

**公式**: P_static = I_leak * V

| 组件 | I_leak 估算 | P_static |
|------|-------------|----------|
| 逻辑 (~500K gates) | ~500 µA | 0.35 mW |
| SRAM (~5KB) | ~100 µA | 0.07 mW |
| **总计** | | **~0.42 mW** |

### 2.3 总功耗

| 类型 | 功耗 | 预算 | 余量 |
|------|------|------|------|
| 动态 | ~22 mW | 50 mW | 56% |
| 静态 | ~0.42 mW | 5 mW | 92% |
| **总计** | **~22.42 mW** | **55 mW** | **59%** |

## 3. 低功耗策略

### 3.1 Clock Gating

| 级别 | 实现 | 条件 |
|------|------|------|
| 模块级 | ICG cell | IDLE 状态 |
| 子模块级 | ICG cell | 子模块空闲 |

**Clock Gating 覆盖率目标**: >= 90%

### 3.2 操作数隔离

| 信号 | 隔离条件 | 实现 |
|------|----------|------|
| MAC 输入 | MAC 空闲时 | AND gate |
| DMA 数据 | DMA 空闲时 | AND gate |

### 3.3 低功耗模式

| 模式 | 触发 | 时钟状态 | 功耗 |
|------|------|----------|------|
| Active | 计算中 | 全开 | ~22 mW |
| Idle | 完成后 | 门控 | ~0.5 mW |
| Sleep | 软件控制 | 全关 | ~0.4 mW |

---

## 4. 功耗优化建议

| 优化 | 预期节省 | 实现难度 |
|------|----------|----------|
| 增加 Clock Gating | 10-20% | 低 |
| SRAM Banking | 5-10% | 中 |
| 电压频率调节 | 20-30% | 高 |
| 操作数隔离 | 5-10% | 低 |
